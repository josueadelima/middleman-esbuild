## Middleman ESBUild + Sass + Uikit

- [Middleman](https://middlemanapp.com)
- [ESBuild](https://esbuild.github.io/)
- [Sass](https://sass-lang.com)
- [Uikit](https://getuikit.com/)

### Installation
```
gem install bundler
brew install yarn
```

```
bundle install
yarn install
```

### Development & Build

```
bundle exec middleman s
bundle exec middleman b
```
